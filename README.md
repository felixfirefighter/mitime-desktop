# MiTime Desktop

## Prerequisites
1. Install NodeJS - https://nodejs.org/en/

## First Step
```
// install dependencies
npm install
```

## Package

```
// Build a package for local platform (i.e. if you're on Mac, it will build a Mac package)
npm run package

// Build for other platform by passing "win", "mac", or "linux"
npm run package -- --win
```
